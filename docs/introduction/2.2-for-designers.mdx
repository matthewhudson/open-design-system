# Getting Started for Designers

<Intro>
This guide covers our Figma implementation strategy and token management workflow.
</Intro>

## Figma Foundation

### Core Setup

1. **Start with the Library**  
    Always begin new projects in our [Figma Team Library](https://www.figma.com/file/...)  
    - Use **Components tab** → "Search widgets" to find approved patterns  
    - Never detach components - use variants instead  

2. **Variables vs. Tokens Studio**  
   We use a hybrid approach:  
   ```plaintext
   Figma Variables → Basic design decisions (spacing, radii)
   Tokens Studio → Complex token relationships (typography, shadows)
   ```
3. **Layer Naming Convention**
    Follow our `category/component/variant` structure:
    - Button/Primary/Hover → ✅
    - Blue clicky thing → ❌
